# 我的量化交易学习记录（Backtrader 系列）

这是我从零开始学习量化交易的代码记录，目前主要使用的是 Python 回测框架 —— **Backtrader**。本项目分为多个每日任务（Day 1 ~ Day 20），每个任务都对应一套策略或功能模块，逐步掌握回测、参数优化、成交量分析、挂单管理等内容。

> 📌 注意：这些代码是**学习笔记性质**，并非实盘策略，也没有语音讲解，主要用于自学记录和分享。

---

## 📁 项目结构

- `Day 01` 到 `Day 20`：对应 Backtrader 的每日学习主题。
- 所有 `.py` 文件均为实战回测脚本，可直接在本地运行。
- 后续将加入 Zipline 的学习内容，与 Backtrader 并列展示。
