Backtrader 学习任务笔记（Day 1 - Day 20）
欢迎来到我的 Backtrader 学习笔记仓库！
本仓库记录了我从 Day 1 到 Day 20 的 Backtrader 学习任务代码和心得总结。内容以学习和练习为主，帮助我逐步掌握量化回测框架的核心功能。



项目背景
Backtrader 是一个功能强大的 Python 量化回测框架，适合学习和测试各种交易策略。
本项目目标不是实盘交易策略，而是系统学习 Backtrader 的基础用法和策略开发流程。



内容介绍
Day 1 - Day 5：
熟悉 Backtrader 环境搭建，数据导入和基础策略写法，掌握简单的买卖信号生成及回测流程。

Day 6 - Day 10：
学习多品种、多时间周期的策略构建，掌握指标计算、数据处理和交易信号管理。

Day 11 - Day 15：
深入订单管理，理解挂单、撤单和订单状态，实现更复杂的交易逻辑。

Day 16 - Day 20：
参数优化和策略性能评估，学习如何批量测试不同参数组合，筛选最优策略配置。



代码结构
Backtrader第1天.py 至 Backtrader第20天.py：每日学习任务代码
data/：历史行情数据（CSV格式）
其他辅助文件和笔记



使用说明
克隆本仓库
安装 Backtrader (pip install backtrader)
运行对应的Day代码文件
结合代码和注释理解策略实现逻辑



学习心得
Backtrader 框架非常灵活，学习曲线陡峭，需要耐心逐步深入
理解订单和交易信号的生命周期是关键
结合参数优化可以有效提升策略表现
实战中需结合更多数据和风控模块，当前代码仅供学习参考
