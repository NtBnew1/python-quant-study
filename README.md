# Alpha Vantage 股票数据抓取脚本

这个项目是我学习量化交易过程中的一个练习，用 Python 调用 Alpha Vantage API 获取某支股票过去一年的每日数据，并保存为 CSV 文件。

👉 搭配我的B站/YouTube频道《我在学量化》视频使用。

使用库：
- pandas
- alpha_vantage

数据来源：
- [Alpha Vantage](https://www.alphavantage.co/)

使用方式：
1. 安装库：`pip install pandas alpha_vantage`
2. 运行脚本，根据提示输入股票代码
